Python Tutorials

Python lists and loops are two important beginner concepts. Lists let you store multiple values in one variable, and loops let you repeat code.

Lists

A list is used to store multiple items in one variable. Lists use square brackets and can hold words, numbers, or other data.

List Example Meaning
fruits = ["apple", "banana", "orange"] Creates a list with 3 items
fruits[0] Gets the first item in the list
fruits.append("grape") Adds a new item to the list 
fruits = ["apple", "banana", "orange"]
print(fruits[0])
fruits.append("grape")
print(fruits)

Loops

Loops are used to repeat code multiple times. Python mainly uses two types of loops: for loops and while loops.

A for loop repeats through a sequence such as a list or range of numbers. 

colors = ["red", "blue", "green"]

for color in colors:
    print(color)

A while loop continues running as long as a condition remains true. 

count = 1

while count <= 3:
    print(count)
    count = count + 1
